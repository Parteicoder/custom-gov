# Custom Governor - Keqing

Dieses Mod fügt **Keqing** als einzigartigen Gouverneur für Civilization VI hinzu.

## Basis

Dieses Projekt ist als Erweiterung / Add-on zur bestehenden **Liyue & Keqing Mod** gedacht:

- https://github.com/dwughjsd/LiyueInazuma_civ6mod
- Steam Workshop: https://steamcommunity.com/sharedfiles/filedetails/?id=2944798624

## Aktueller Inhalt

- **Keqing - Yuheng der Sieben Sterne** als Gouverneur
- Vollständiges Promotion-Tree mit 4 Stufen
- Starke Synergie mit Handel, Ressourcen und dem Goldenen Haus

## Geplante Erweiterungen

- Zweiter Gouverneur: Ningguang
- Icons
- Weitere Balancing-Anpassungen

## Installation

1. Lade den Ordner herunter
2. Kopiere ihn in deinen `Mods` Ordner
3. Aktiviere die Mod im Spiel
